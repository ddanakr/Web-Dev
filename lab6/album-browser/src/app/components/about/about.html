Lab: 6 - Album Browser

In this lab, I built an Album Browser application using Angular's Router and HttpClient modules. The application fetches data from a public REST API and features a service layer to manage data flow. Key focuses included:

  • Multi-page SPA: Implementing seamless navigation between Home, About, and Album views.
  • Routing & Parameters: Using dynamic routes to pass IDs and display specific album details or photos.
  • HTTP & Observables: Handling asynchronous data requests and updating the UI reactively.
  • CRUD Operations: Enabling users to view, edit, and delete album entries.
